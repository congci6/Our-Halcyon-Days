**致谢**

　　在几天前，我就想写点东西，想写个简单的致谢挂在论文的末尾，但一直没有动笔去写。

　　今天，我终于再次戴上耳机，坐在宿舍桌前的座位旁，像几年前一样，再次写下这些简单的文字。

　　大学这几年，的确改变了许多人，它带来了价值观冲击，但也改变着个人对理想寻求与达成的思考，或许是它的确影响了我几年青春，才让我觉得值得纪念，尽管现在无法回归于最初起点。

　　起初，心血来潮，在大学志愿选择了不少未曾听闻的专业，最后兜兜转转来到了大泽湖畔的网安专业，没有进入我想要的领域，也没有完成当时的稚嫩梦想。

　　后来，就索性没管了，也就渐渐遗忘了。

　　九月初，随着高校的陆续报到，大多数人都前往学校，我也不例外，人群熙熙攘攘，总想着要做些什么，有幸与几个志同道合的同龄人同行，走走停停，最后，也相互帮助扶持，走完了人生最迷茫的几年，感谢我的室友兄弟们，彼此陪伴，守望相助。

　　当然，也要感谢我的家人，没有他们的托举与理解，我无法平静幸福地度过我的大学时光。

　　说实话，我并不是一个外向开朗的人，就干脆很少关注课堂互动，感谢大学阶段的所有老师的传道、授业、解惑。在此，特别感谢胡建兵老师在论文上的悉心指导与帮助，谨致谢忱，学生朽木。

　　大学期间，我才开始接触计算机行业，学了不少网课，接触了不少方向，不过，也没有做出什么名堂来，零零散散几个可怜的奖项。跌跌撞撞，磕磕绊绊，走完了或许是人生最好的时期。

　　不过，渐渐我开始规律我的生活，也学会了坦然接受失败和痛苦。理想的寻求与达成，终究回归于自我的原点，对自我思索与发掘，对自我的革新与创造。敬自己，也敬真挚而热烈的青春。

　　写到这里，我结束了全部的本科学业，大学四年，恍然如梦。祝来年春，繁花似锦，不再自否。
